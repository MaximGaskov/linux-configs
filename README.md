# usage
```bash
ansible-playbook -i playbooks/inventory playbooks/main.yml
```
## requirements:
* python
* ansible

## Non-standard key-bindings:
keys | action
------------ | -------------
cmd+home | reboot/poweroff/logout menu
cmd+t | Thunat file manager (opens on 4th workspace)
cmd+i | Firefox (2nd workspace)
